# dotfiles

🏠.dotfiles

-- zsh, nvim, kitty, lazygit ...

# Requirements

Required: git, zsh, xclip, file, ripgrep, fzf, fd, zoxide, chafa.

Optional: neovim, yazi, kitty, lazygit, tmux, etc.

# Install

Jsut

```bash
git clone https://github.com/hnpyn/dotfiles.git
cd dotfiles
chmod u+x install.sh
./install.sh
```

or you can link what you like, for example

```
ln -s /the/path/of/dotfiles/config/nvim ~/.config/
```
